## API List

- 등록된 메뉴 리스트 가져오기(GET)
  ```
    /menu
  ```

### 주문자

- 메뉴 주문 가능

### 피주문자

- 메뉴 등록(POST)
  
  ```
    /menu
  ```

- 메뉴 삭제(DELETE) : 메뉴의 id값을 파라미터로 받아 삭제하는 기능

  ```
    /menu/:id
  ```

- 메뉴 수정(PUT)

  ```
    /menu/:id
  ```

- 메뉴 상태 조회(GET)

  ```
    /orderlist
  ```

## API List

### 주문자
- 메뉴 리스트 가져오기 
- 메뉴 주문 가능


### 피주문자

- 메뉴 등록(GET)
  
  ```
    /menu
  ```
- 메뉴 삭제(DELETE)
  ```
    /menu/:menu
  ```
- 메뉴 수정(PUT)
  ```
    /menu
  ```
- 메뉴 상태 조회(GET)
  ```
    /orderlist
  ```
